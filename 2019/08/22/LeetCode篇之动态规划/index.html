#198.大家劫舍 1

题目：你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。给定一个代表每个房屋存放金额的非负整数数组，计算你在不触动警报装置的情况下，能够偷窃到的最高金额。

示例：

解：

定义 dp 数组用来存储最大的抢劫量，其中 dp[i] 表示抢到第 i 个住户时的最大抢劫量。

由于不能抢劫邻近住户，如果抢劫了第 i -1 个住户，那么就不能再抢劫第 i 个住户，所以

dp[i]=max(dp[i-2]+nums[i],dp[i-1])

思路：

将问题分解，当只有第一行时，只有一种走法即是左到右

再往下逐渐添加第二行进行比较，每次都可看做是四个数中，左上角的数如何最优地走到右下角，直到第二行末尾，得出前面两行的最优走法

在最优解的基础上，跟第三行比较，再得出前面三行的最优走法即最优解

即除了第一行，每次都是两行比较，把第二行最后一个数当做终点，以此类推

用一个数组边计算比较边存目前最优解 int[] dp = new int[grid[0].length]

区域和检索-数组不可变

#303.区域和检索-数组不可变

题目：给定一个整数数组 nums*，求出数组从索引 *i 到 j (i ≤ j) 范围内元素的总和，包含 i, j 两点。

示例：

解：

求区间 i ~ j 的和，可以转换为 sum[j + 1] - sum[i]，其中 sum[i] 为 0 ~ i - 1 的和。

这样可以内置一个总和数组，每个元素的值为坐标小于它的元素的总和，即0 ~ i - 1

所以，这个数组长度为传入数组的长度+1
